#  typster 

 A desktop application for a new markup-based typesetting language, [typst](https://github.com/typst/typst).
Typster is built using [Tauri](https://tauri.app/).


# features
- [x] auto code completion
- [x] export PDF file 
- [x] error tip
- [x] use latest typst v0.12
